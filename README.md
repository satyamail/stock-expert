# StockExpert 📈💎

StockExpert is an intelligent, high-fidelity system built with rich glassmorphic visuals, state-of-the-art interactive charts, and real-time financial tracking algorithms. It is designed to evaluate fundamental health, gauge market mood, summarize quarterly earnings, analyze technical momentum ("momos"), graph projections, and simulate automated stock executions.

---

## Key Features

1. **Intelligent Fundamental Screener**: Dynamic evaluation scores computed based on Debt-to-Equity, Return on Equity (ROE), Price-to-Earnings (P/E) ratios, and margin thresholds. Includes a safety discount calculator based on Calculated Intrinsic Value vs. Market Price.
2. **Market Mood Index (Greed & Fear)**: Visually stunning interactive gauges illustrating live technical breadth, news sentiment, corporate performance, and market volatility.
3. **Advanced Stock Analyzer**:
   - **Performance Timeline**: 6-month historical stock charts using Chart.js.
   - **Quarterly Earnings Analyzer**: Compares sequential and year-over-year revenue, EPS, and profit margins.
   - **Future Projections**: Graphs multi-year projected revenue & net income.
   - **Momo Audit**: Provides momentum calculations (RSI, MACD alignment, moving averages) and dynamic pros & cons reports.
4. **Mock Transaction Broker & Portfolio Tracker**: Safe sandboxed sandbox simulating real-time buy/sell executions. Computes average costs, manages cash balances, plots portfolio allocation distributions, and tracks profit-and-loss.

---

## Technical Stack & Dependencies

- **HTML5 & CSS3**: Custom modern dark theme with HSL colors, responsive design systems, and glassmorphic aesthetics.
- **Vanilla ES6 JavaScript**: Performant calculations, state management, search suggestions autocomplete, and transaction handling.
- **Chart.js (CDN)**: Interactive tooltips and gorgeous finance-focused charts.
- **Lucide Icons (CDN)**: Premium modern developer icons.
